------------------------------------------------OmniTerminal----------------------------------------------------------------
此项目旨在实现一个多功能工具类应用程序，我们需要你的各种建议以帮助我们增加各种功能和改进程序。
在GitHub上提交你的想法：https://github.com/zghzd/OmniTerminal
------------------------------------------------环境依赖------------------------------------------------------------------------
64位操作系统。
你需要安装vc++运行库，建议安装最新版本。
之后，只要运行OmniTerminal.exe即可。
---------------------------------------------更新/回滚/修复功能-----------------------------------------------------------------
关于更新/回滚/修复功能：
	我们通过提前预测新版本更改的内容，将其打包至包中。当用户使用包时，程序会替换所需文件。
    因此，如果用户的程序不完整，更新/回滚操作就会发生错误。
更新/回滚/修复 功能所需的先决条件：
	更新/回滚/修复 功能需要对应的 更新包/回滚包/修复包 ，版本必须在1.0.0及以上。
如果你回滚到了1.0.0以前的版本，你可能无法使用 更新/回滚/修复 功能。
指令：”install_update““install_fix”“install_back”

关于“包”：
    “包”是指一个文件夹，包的第一级目录中必须包含“_package”文件。
    用户收到的包可能是一个压缩文件，此时，用户需手动解压，并确保解压后的所有文件位于~\resource\<update或fix或back>\的文
件夹内。
